# moi-akkugyar-terkep Interaktív térkép a magyarországi akkumulátoripari **üzemekről, beszállítókról és kapcsolódó létesítményekről**. Nem “random pöttyök”: **rétegek szerint** (cellagyár, vegyipar/alkatrész, összeszerelés, beszállítók, hulladék, egyéb). ## Mit tud? - ✅ **Rétegek (Layer control)**: - 🔴 cellagyár (akku-cella) - 🟠 alkatrész / vegyipar (katód, szeparátor, elektrolit, rézfólia) - 🟡 modul / összeszerelés / rendszer - 🟢 kisebb beszállítók / alkatrészgyártás - 🔵 újrahasznosítás / hulladék - ⚫ egyéb kapcsolódó helyek - ✅ **Popup** minden ponthoz: cég + település + megjegyzés - ✅ **Marker cluster**: sok pontnál sem lesz káosz - ✅ **GitHub Pages-kompatibilis**: statikus HTML, csak megnyitod és megy ## Demo (GitHub Pages) 1. Repo → **Settings** → **Pages** 2. Source: **Deploy from a branch** 3. Branch: **main** / folder: **/(root)** 4. Mentés után elérhető lesz: `https://.github.io/moi-akkugyar-terkep/` ## Futtatás helyben Semmi varázslat: - Nyisd meg a `index.html` fájlt böngészőben **vagy** - Indíts egy egyszerű local szervert: - Python: ```bash python -m http.server 8000 ``` majd: `http://localhost:8000` ## Fájlok Ajánlott minimum struktúra: - `index.html` – a térkép (Leaflet + rétegek + pontok) - `README.md` – ez a leírás Ha később tisztábbra akarod: - `data.json` – pontok külön fájlban - `app.js` – logika külön fájlban - `style.css` – stílus külön fájlban ## Adatok módosítása Az `index.html`-ben a `DATA` tömbben vannak a pontok.
